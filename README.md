# misis_table
## Скачивание проекта
```sh
git clone https://github.com/ngumbk/misis_table.git
```

## Установка зависимостей
### С venv
```sh
virtualenv venv
./venv/Scripts/activate
pip install -r requirements.txt
```

### Без venv
```sh
pip install -r requirements.txt
```

## Запуск скрипта
```sh
python misis_table.py
```
